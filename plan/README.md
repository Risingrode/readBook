# “边读边玩” 记忆强化阅读器 - 规划总览 (Master Plan Index)

本文档是本项目从“极简阅读器”向**“高留存率、神经级沉浸式记忆学习平台”**转型的全部核心设计蓝图。请开发人员在修改代码前，仔细阅读以下规划。

## 📁 规划目录与阅读顺序

### 📍 1. 基础审查与总任务清单
- [x] **`tasks.md`** (总调度中心)
  - 描述：当前规划阶段的所有任务历史与整体路线图。
- [x] **`project_review.md`** (现有技术基建)
  - 描述：对当前项目的 Next.js/Prisma 架构分析以及初始功能的摸底报告。

### 🎮 2. 游戏化与记忆设计 (Gamification & Memory)
- [x] **`gamification_design.md`** (核心循环)
  - 描述：初步阐述了将阅读转化为“游戏经验值 (EXP)”、成长系统和成就激励的宏观概念。
- [x] **`memory_game_reader.md`** (动感记忆阅读器交互)
  - 描述：详细设计了“PPT幻灯片分块阅读”、“会跑的文字捕捉”和“记忆消消乐”等直接作用于阅读界面的交互。

### 🧠 3. 心理学留存机制 (Psychological Retention)
- [x] **`psychology_retention_mechanics.md`** (成瘾理论模型)
  - 描述：深度剖析蔡加尼克效应、斯金纳箱、艾宾浩斯遗忘曲线等认知心理学原理如何应用于本阅读器以提升用户留存。
- [x] **`retention_systems_design.md`** (留存系统落地蓝图)
  - 描述：将上述心理学理论转化为具体的软件功能模块，如：随机掉落暴击系统、章节悬念打断、盘前热身战和“生命之火”。

### 🛠️ 4. 架构重构指南 (Architecture & UI)
- [x] **`ui_architecture_rework.md`** (UI动效与数据库)
  - 描述：为实现动感文字和各项游戏化功能，建议采用的动画库 (`framer-motion`) 以及对现有 `schema.prisma` (`UserStats`) 的扩展建议。

### 🔥 5. 终极进化方向 (The Ultimate Vision)
- [x] **`extreme_upgrade_specs.md`** (极限神经级体验)
  - 描述：突破常规的激进设计。包含：基于眼动追踪的视线锁定、AI实时造境与NPC伴读、黑暗森林社交掠夺法则，以及烧毁未掌握知识的极端惩罚机制。
