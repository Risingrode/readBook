# 动感与重构规划 (UI & DB Rework Specs)

## 1. 动感文字交互 (Kinetic Typography)
- **技术选型:** 
  - React 动画库 `framer-motion` 用于处理卡片翻转、UI 弹窗。
  - WebGL / Canvas 或轻量物理引擎 `matter.js` 处理文字的“重力”、“碰撞”和“掉落”效果。
- **交互场景:**
  - **受惊的文字:** 当读到描写“恐惧”、“剧烈”的段落时，个别字词会产生高频微小震动。
  - **消散 (Fade away):** 读完的句子像沙子一样随风飘散，强迫用户的视觉焦点只能留在当前正在读的字上。

## 2. 数据库扩展建议 (Prisma Schema Updates)
基于以上规划，未来的 `schema.prisma` 需要做如下扩充（仅规划，不修改现有代码）：
- **`UserStats` 模型扩展:**
  - 添加 `exp` (Int): 累计经验值。
  - 添加 `level` (Int): 当前等级。
  - 添加 `flameHealth` (Int): 火种健康度 (0-100)。
- **新增 `MemoryNode` 模型 (记忆节点):**
  - 记录用户捕捉到的关键词、释义、上次复习时间 (对应遗忘曲线) 以及熟练度。
- **新增 `Inventory` 模型 (背包):**
  - 存储用户掉落的“护盾”、“主题”等道具。
